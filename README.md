# 🎓 Academia-POO

Proyecto final integrador del **Módulo 2 - Unidad 1** del programa *Java Senior AI*.

## 📌 Descripción

Este proyecto consiste en el desarrollo de un sistema orientado a objetos para la gestión de una academia. Se aplican conceptos fundamentales de la Programación Orientada a Objetos (POO) como:

* Encapsulamiento
* Herencia
* Polimorfismo
* Abstracción

El sistema permite modelar entidades clave como estudiantes, cursos y procesos administrativos básicos.

## 🎯 Objetivos

* Aplicar buenas prácticas de desarrollo en Java
* Implementar estructuras de clases bien organizadas
* Simular un sistema real de gestión académica

## 🛠️ Tecnologías utilizadas

* Java
* Programación Orientada a Objetos (POO)
* Git & GitHub

## 📂 Estructura del proyecto

* `src/`: Código fuente , clases

## 🚀 Cómo ejecutar

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/Angel-D28/Academia-POO.git
   ```

2. Abrir el proyecto en tu IDE (IntelliJ, Eclipse, etc.)

3. Ejecutar la clase principal (`Main.java`)
